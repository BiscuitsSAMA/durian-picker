# 🍈 挑个榴莲吧

挑榴莲专家 Skill——任何支持 Skill 的 AI Agent（Claude Code、Hermes、小龙虾、Minis 等）均可使用。

## 使用方法

### 💬 聊天模式（跨平台通用）

直接发榴莲照片 + 回答三个问题：
1. 什么品种？（金枕头/猫山王/干尧/托曼尼/黑刺/不知道）
2. 摇起来有声音吗？（有撞击声/没声音/没试）
3. 捏相邻刺能捏动吗？（能捏动/捏不动/没试）

Agent 加载 SKILL.md 后会自动按四步法+多模态能力分析，给出结论。

### 🖥 前端页面（Minis 环境增强版）

```bash
# 启动服务
nohup python3 server.py > /tmp/durian_server.log 2>&1 &

# 浏览器打开
# http://127.0.0.1:8899/
```

打开页面后拍照上传 + 点选选项 → 本地调 VL 模型真正分析图片 → 直接出结论。

## 分析逻辑

| 维度 | 权重 | 干包✅ | 湿包🟡 | 生包❌ |
|------|------|--------|--------|--------|
| 果型 | 30% | 偏圆、底平 | 圆/长均可 | 不规则、尖底 |
| 刺 | 25% | 尖硬密集 | 钝圆有弹性 | 紧实捏不动 |
| 声音 | 25% | 撞击感/沙沙声 | 闷声/无声 | 完全无声 |
| 捏刺 | 20% | 微有弹性 | 偏软易捏 | 硬如石头 |

## 量化判断

| 维度 | 权重 | 干包✅ | 湿包🟡 | 生包❌ |
|------|------|--------|--------|--------|
| 果型 | 30% | 偏圆、离心率<1.3 | 圆/长均可 | 不规则、离心率>1.5 |
| 刺 | 25% | 尖硬密集、奶油尖 | 钝圆有弹性 | 紧实捏不动、细长 |
| 声音 | 25% | 撞击感/沙沙声 | 闷声/无声 | 完全无声 |
| 捏刺 | 20% | 微有弹性 | 偏软易捏 | 硬如石头 |

## 品种速查

| 品种 | 特征 |
|------|------|
| 金枕头🥇 | 金黄、会裂口、甜糯、新手友好 |
| 猫山王👑 | 黄绿、不裂需刀开、苦甜浓郁 |
| 黑刺🖤 | 偏绿不裂、奶油质地、顶级 |
| 干尧🫒 | 青绿、微裂、细腻高甜 |
| 托曼尼🟢 | 偏绿不裂、似猫山王籽大 |
